# Example Skill

## Purpose

This is a demonstration skill showing how to structure a skill for the agent.

## Usage

When users ask about the example skill or need to perform example tasks, this skill provides guidance.

## Key Features

- Demonstrates proper SKILL.md format
- Shows how to structure content
- Provides example guidance for the agent

## Notes

Skills are loaded from the `skills/` directory and can be accessed via the `get_skill` tool.
